1. 시험부터 보자
1차시험은 1시간 30분 2차시험은 4시간정도 주어진다.
1차 시험은 예전에 과제로 내준 txt 파일보단 훨신 쉽고 객관식으로 문제가 출제될 거니 한번씩 풀어보는 것을 권한다. (모르면 구글링을하거나 물어보시오)

2. 외부평가 실기 문제
각 개발도구는 따로 준비를 해두어 올려줄거니 없으면 받으러 오시오. (설치및 재설치는 알아서)


3. 켜지자 마자 작업 설정 부터 해준다. Preferences 에서 files 검색 후 css,html,jsp 파일 UTF-8로 바꿔준다.

4. Dynamic web Project 를 만든 후
프로젝트명을 우클릭 - New - Other로 들어가서 Server를 찾는다.





5. Java Build Path에 서버 라이브러리(Tomcat)을 추가한다
Project Facets에서 맞는 Java 버전에 체크하고 오른쪽의 Runtimes에서 서버도 체크해준다


Java 버전을 사용하는 버전에 체크하고
Server에서 구동할 서버 선택후 Apply해준다.

Server연결이 확인되었으면 아래쪽에서 Server를 더블클릭 한다.

양식에 맞게 Server 설정해 준다.(포트번호도 같이)

Edit를 이용해 경로와 오토리로드 등 세부 설정을 한 후 Ctrl+S로 저장한다.

6. 위와 같이 JSP 파일을 만든 후 Server를 돌려본다.
